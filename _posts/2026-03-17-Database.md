# 📖오늘 배운 내용 

---

* 파이썬을 설치했다. 
* 파이썬으로 패키지를 설치하고, 연동하는 방법을 배웠다. 

```python
# MySQL 연동 
import mysql.connector

# 1. DB 연결
conn = mysql.connector.connect(
    host="localhost",
    user="root",
    password="본인비밀번호",
    database="My_Database"
)
cursor = conn.cursor()

# 2. 데이터 조회
cursor.execute("SELECT * FROM students2 WHERE grade = 2")
rows = cursor.fetchall()

# 3. 결과 출력
for row in rows:
    print(row)

# 4. 종료
cursor.close()
conn.close()
```

### 💡내가 오늘 작업간 느낀점. 
- 데이터관리도 Python을 활용하여 효율적으로 관리해야 한다는걸 알게 되었다.
