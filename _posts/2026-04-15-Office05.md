# 📝 자료 분석·가공

>**과목:** 1과목. 사무자동화시스템  
**단원:** 2. 자료 관리 > 2. 자료 분석·가공  
**학습 목표:** 수집된 자료에서 유의미한 정보를 도출하기 위한 통계분석의 기초 개념을 이해하고, 실무에 사용되는 주요 분석 도구의 종류를 파악한다.

---

## 📍 1. 통계분석 개요

수집된 대량의 데이터를 요약하고, 데이터 속에 숨겨진 규칙이나 패턴을 찾아내기 위해 수학적/통계적 기법을 적용하는 과정입니다.

### 1.1) 기초 통계량 (기술 통계)    
데이터의 전반적인 형태와 특징을 요약하여 설명하는 가장 기본적인 분석 방법입니다.
* **중심 경향성:** 데이터가 어느 위치에 모여 있는지를 나타내는 값입니다. (예: `평균(Mean)`, `중앙값(Median)`, `최빈값(Mode)`)
* **산포도:** 데이터가 중심을 기준으로 얼마나 흩어져 있는지를 나타내는 값입니다. (예: `분산(Variance)`, `표준편차(Standard Deviation)`)

### 1.2) 자료 가공의 주요 단계
분석 도구에 데이터를 넣기 전, 품질을 높이는 전처리(가공) 과정이 필수적입니다.
* **데이터 정제 (Cleaning):** 결측치(비어있는 값)를 채우거나, 이상치(터무니없이 크거나 작은 값), 중복된 데이터를 제거하여 오류를 바로잡습니다.
* **데이터 변환 (Transformation):** 서로 다른 단위(예: 원, 달러)를 통일하거나, 분석 모델에 맞게 데이터의 형태를 변경합니다.

---

## 📍 2. 자료 분석·가공 도구

자료의 규모와 분석의 깊이에 따라 다양한 소프트웨어 도구가 사용됩니다.

### 2.1) 스프레드시트 (Spreadsheet)
* **대표 프로그램:** Microsoft Excel (엑셀), Google Sheets 등
* **특징:** 표 형태로 데이터를 직관적으로 입력하고 다룰 수 있으며, 내장된 함수와 피벗 테이블(Pivot Table)을 이용해 빠르고 쉬운 기초 분석 및 가공이 가능합니다. (사무자동화의 가장 핵심적인 도구)

### 2.2) 통계 전용 패키지 소프트웨어
* **대표 프로그램:** SPSS, SAS 등
* **특징:** 전문적인 통계 분석을 위해 개발된 상용 소프트웨어로, 복잡한 통계 수식을 직접 코딩하지 않아도 UI 메뉴 클릭만으로 고급 회귀 분석이나 분산 분석 등을 수행할 수 있습니다.

### 2.3) 프로그래밍 언어 기반 분석 도구
* **대표 언어:** R, Python
* **특징:** `R`은 통계 계산과 그래픽 시각화에 특화된 오픈소스 언어입니다. `Python`은 Pandas, NumPy 등의 라이브러리를 활용하여 대용량 데이터의 가공과 머신러닝 분석에 매우 강력한 성능을 발휘합니다.

### 2.4) BI (Business Intelligence) 시각화 도구
* **대표 프로그램:** Tableau(태블로), Power BI 등
* **특징:** 가공된 데이터를 바탕으로 실시간 대시보드(그래프, 차트)를 구성하여 시각적으로 정보를 전달하는 데 특화된 의사결정 지원 도구입니다.
