## 📌 1. 2026년 시험 일정 및 기본 정보

* **🔗 시험 신청 링크:** [큐넷(Q-Net) 공식 홈페이지 바로가기](https://www.q-net.or.kr)
<br>
* * **📅 2026년 정기 2회차 공식 시험 일정**
  * **필기 원서 접수:** 2026. 04. 20(월) ~ 04. 23(목)
  * **필기 시험 일자:** 2026. 05. 09(토) ~ 05. 29(금)
  * **필기 합격자 발표:** 2026. 06. 10(수)
  * **실기 원서 접수:** 2026. 06. 22(월) ~ 06. 25(목)
  * **실기 시험 일자:** 2026. 07. 18(토) ~ 08. 05(수)
  * **최종 합격자 발표:** 2026. 09. 11(금)
<br>
* **📝 평가 방식 및 합격 기준**
  * **필기:** 총 60문항 (과목당 20문항), 제한 시간 1시간 30분.
    * 전 과목 평균 60점 이상 합격 (단, 한 과목이라도 40점 미만이면 과락).
  * **실기:** 엑셀, 액세스, 파워포인트 통합 작업형, 제한 시간 2시간. 
    * 100점 만점 기준 60점 이상 합격 (실격 사유 주의).

## 🗺️ 2. 학습 로드맵

시험까지 남은 기간을 최대한 효율적으로 쓰기 위한 4단계 전략입니다.

1.  **[이론 뼈대 잡기]** 아래의 체크리스트를 따라 과목별 빈출 개념을 빠르게 훑습니다.
2.  **[코드 흐름 파악]** 2과목의 C, Java 코드와 1과목의 SQL 쿼리문은 직접 눈으로 결과를 도출하는 연습을 합니다.
3.  **[기출 무한 반복]** CBT 기출문제를 풀며 오답 트러블슈팅 아카이브를 작성합니다.
4.  **[실기 대비 연계]** 필기를 공부할 때 등장하는 엑셀 함수나 액세스 쿼리 개념을 실기 작업과 연결 지어 이해합니다.

## ✅ 3. 학습 체크리스트

과목별 이론 학습 페이지의 목차와 1:1로 완벽하게 대응되는 체크리스트입니다. 학습을 완료한 세부항목의 괄호 안을 `[x]`로 채워가며 진도를 관리하세요.

### 📖 [1과목] 사무자동화시스템
- **1. 개발자 환경 구축**
  - [x] 1.1 운영체제 기초 활용 (개념, 환경 설정, 종류, 명령어)
  - [x] 1.2 기본 개발환경 구축 (도구 종류, 프로그래밍 도구 및 패키지 설치)
- **2. 자료 관리**
  - [ ] 2.1 자료 수집 (정형/비정형 특성 파악)
  - [ ] 2.2 자료 분석·가공 (통계 분석 개요)
- **3. 사무자동화프로그램 활용**
  - [ ] 3.1 스프레드시트 활용 (셀 참조, 수식, 함수, 필터, 매크로)
  - [ ] 3.2 데이터베이스 활용 (스키마, 테이블, 쿼리, 폼/보고서)
  - [ ] 3.3 프레젠테이션 활용 (디자인, 애니메이션 효과)
- **4. SQL 작성**
  - [ ] 4.1 데이터 조회 (Select, 테이블 조인, 집합연산자)
  - [ ] 4.2 데이터 수정 (DDL, DML, DCL, 무결성)
- **5. HW 단말장치 운영관리**
  - [ ] 5.1 단말장치 운용 (주변장치, 드라이버)
  - [ ] 5.2 단말장치 장애 처리 (백업/복구, 로그 관리)

- **1. 프로그래밍 언어 활용**
  - [ ] 1.1 구조적 프로그래밍언어 활용 (C언어 문법, 제어문, 배열)
  - [ ] 1.2 객체지향 프로그래밍 언어 활용 (Java 특성, 캡슐화/상속/다형성)
  - [ ] 1.3 스크립트 활용 (Python, JS 특성 및 동적 타이핑)
- **2. 프로그래밍 언어 응용**
  - [ ] 2.1 언어특성 활용 (언어별 장단점 비교)
  - [ ] 2.2 라이브러리 활용 (가져오기 및 배포)

  - [ ] 1.2 미들웨어 기초 활용 (미들웨어 개념)
