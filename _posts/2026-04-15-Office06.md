# 📝 스프레드시트 프로그램 활용

>**과목:** 1과목. 사무자동화시스템  
**단원:** 3. 사무자동화프로그램 활용 > 1. 스프레드시트 프로그램 활용  
**학습 목표:** 스프레드시트의 기본 데이터 입출력부터 수식, 함수, 데이터 분석 기능(정렬/필터) 및 매크로 활용법을 파악한다. (필기 및 실기 시험의 핵심)

## 📍 1. 데이터 입출력

스프레드시트(엑셀 등)에서 데이터를 올바르게 입력하고 형식을 지정하는 것은 모든 작업의 기초입니다.

### 1.1) 데이터의 종류와 기본 정렬
* **문자 데이터:** 한글, 영문, 특수문자 등. 기본적으로 셀의 **왼쪽**에 정렬됩니다. (숫자를 문자로 입력하려면 숫자 맨 앞에 작은따옴표 `'`를 붙입니다.)
* **숫자 데이터:** 0~9, +, -, %, 콤마(,) 등. 기본적으로 셀의 **오른쪽**에 정렬됩니다.
* **날짜/시간 데이터:** 날짜는 하이픈(-)이나 슬래시(/)로 구분(예: 2026-04-15)하고, 시간은 콜론(:)으로 구분합니다. 기본적으로 셀의 **오른쪽**에 정렬됩니다.

### 1.2) 표시 형식 (셀 서식)
* 입력된 데이터의 '실제 값'은 변하지 않으면서 화면에 '보이는 형태'만 바꾸는 기능입니다. 
* **사용자 지정 기호:** * `#`: 유효한 자릿수만 표시 (0은 표시하지 않음)
  * `0`: 유효하지 않은 자릿수라도 무조건 0으로 표시
  * `@`: 문자 데이터를 표시할 때 사용

## 📍 2. 셀 참조와 수식 (★ 매우 중요)

수식을 다른 곳으로 복사할 때 참조하는 셀 주소가 어떻게 변하는지 이해하는 것은 필기와 실기 모두에서 가장 중요합니다.

### 2.1) 상대 참조 (Relative Reference)
* **형태:** `A1`
* **특징:** 수식을 다른 셀로 복사하거나 자동 채우기를 할 때, 복사한 방향과 이동한 칸 수만큼 **참조하는 셀 주소도 자동으로 변하는** 방식입니다. (기본값)

### 2.2) 절대 참조 (Absolute Reference)
* **형태:** `$A$1` (단축키: `F4`)
* **특징:** 열 문자나 행 번호 앞에 달러($) 기호가 붙어, 수식을 어디로 복사하더라도 **참조하는 셀 주소가 절대 변하지 않고 고정**되는 방식입니다. (환율, 할인율 등 고정값을 참조할 때 사용)

### 2.3) 혼합 참조 (Mixed Reference)
* **형태:** `$A1` (열만 고정) 또는 `A$1` (행만 고정)
* **특징:** 행이나 열 중 하나만 고정하는 방식입니다. 구구단 표처럼 행과 열이 교차하는 수식을 한 번에 작성할 때 유용합니다.

## 📍 3. 주요 함수

함수는 특정 계산이나 작업을 자동으로 수행해 주는 명령어입니다.

### 3.1) 통계 및 수학 함수
* `SUM(범위)`: 지정한 범위의 합계 
* `AVERAGE(범위)`: 지정한 범위의 평균
* `RANK.EQ(내값, 전체범위, 옵션)`: 전체 범위에서 내 값의 순위를 구합니다. (주의: 전체 범위는 반드시 절대 참조 `$전체$범위`로 고정해야 수식을 복사해도 오류가 나지 않습니다.)

### 3.2) 논리 및 찾기/참조 함수
* `IF(조건, 참일 때 결과, 거짓일 때 결과)`: 지정한 조건에 맞으면 참 값을, 틀리면 거짓 값을 반환합니다.
* `VLOOKUP(찾을값, 참조범위, 열번호, 옵션)`: 표의 맨 첫 번째 열에서 특정 값을 찾은 후, 같은 행에 있는 다른 열의 데이터를 가져오는 기능입니다.

## 📍 4. 정렬과 필터

방대한 데이터 중에서 원하는 조건에 맞는 데이터만 추려내거나 순서대로 나열하는 기능입니다.

### 4.1) 정렬 (Sort)
* 특정 열을 기준으로 오름차순(가나다순, 작은 수 ➔ 큰 수) 또는 내림차순으로 데이터를 재배열합니다.
* 여러 개의 정렬 기준을 추가하여(예: 1차 기준 부서명, 2차 기준 직급) 세밀하게 정렬할 수 있습니다.

### 4.2) 자동 필터 (Auto Filter)
* 표의 제목 줄에 드롭다운 화살표를 만들어, 간단한 조건에 맞는 데이터만 화면에 표시하는 기능입니다.
* 여러 열에 필터를 적용하면 항상 두 조건을 모두 만족해야 하는 'AND(그리고)' 조건으로 결합됩니다.

### 4.3) 고급 필터 (Advanced Filter)
* 복잡한 조건(OR 조건 등)을 별도의 셀 영역에 직접 입력하여, 조건에 맞는 데이터를 다른 위치로 복사하거나 추출할 수 있는 강력한 기능입니다.

## 📍 5. 테이블 및 차트

숫자로 된 데이터를 시각적인 그래픽으로 표현하여 추세나 비율을 쉽게 파악하도록 돕습니다.

### 5.1) 주요 차트의 종류
* **세로 막대형:** 항목 간의 값을 직접 비교할 때 가장 널리 쓰입니다.
* **꺾은선형:** 시간에 따른 데이터의 연속적인 변화(추세)를 보여줄 때 적합합니다.
* **원형:** 전체에 대한 각 항목의 '비율'이나 '기여도'를 나타낼 때 사용합니다. (하나의 데이터 계열만 표현할 수 있습니다.)

## 📍 6. 매크로 (Macro)

반복적이고 규칙적인 작업을 일련의 순서대로 기록해 두었다가, 한 번의 클릭이나 단축키로 자동 실행하는 기능입니다.

### 6.1) 매크로의 주요 특징
* 스프레드시트 내부적으로 **VBA (Visual Basic for Applications)**라는 프로그래밍 언어 코드로 기록됩니다.
* 매크로의 이름은 반드시 문자로 시작해야 하며, 공백이나 특수문자는 이름에 사용할 수 없습니다.
* 매크로가 포함된 문서를 저장할 때는 반드시 '매크로 사용 통합 문서(예: `.xlsm`)' 형식으로 저장해야 합니다.
